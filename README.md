# The-World-Incremental
The World Incremental is a browser-based incremental game currently in early development.

The game will begin on Earth in the Overworld layer, with Power as its primary currency. Additional currencies, upgrades, mechanics, and progression layers will be designed and added gradually throughout development.

**Project Status**
Current status: Planning and initial setup
The game is not yet playable. Its core gameplay, progression, balancing, interface, and visual design are still being developed.

**Planned Features :**
- Incremental Power generation
- Purchasable upgrades
- Automatic resource generation
- Offline progress
- Local save system
- Large-number formatting
- Multiple currencies
- Multiple progression systems
- Multiple world layers
- Responsive browser interface

The planned features may change as development continues.

**Development**
This repository will contain the source code and development history for The World Incremental.

**Version**
Current development stage: Pre-Alpha
